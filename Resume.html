Portfolio

Currently, I am working on an IoT-based Active Plant wall for intelligent indoor Climate control. The System consists of a local unit comprising edge nodes with sensors and actuating units. Sensor nodes are deployed and collecting data real-time data to sent to the Fog node. Fog node consists of the micro-processor Raspberry Pi 3. Micro-processor acts as a gateway between edge nodes and Cloud. Data filteration is applied on fog node and further data packets are sent to Cloud. For Cloud unit we are utilising Azure Cloud Platform by MicroSoft. For Data Storage we have integrated Azure Cosmos DB with Raspberry Pi and for remote monitoring we have utilised Azure IoT Central Application.
